### 🌟 4分：惊艳/卓越表现

**打分原则**：4分是千里挑一（<10%），需同时满足所有升级条件。

**"极其突出"4重测试**：

2. **惊喜测试**：用户看到会说"哇"还是"嗯"？

## 4. 打分流程

### 流程图

```
检查0分红线 → Hit? → 直接0分，结束
    ↓ Miss
检查1分条款 → Hit? → 直接1分，结束
    ↓ Miss
检查2分条款 → Hit? → 判2分（检查升档条件）
    ↓ Miss
确认3分基础达标 → 默认3分
    ↓
检查4分条件 → 全满足? → 4分
    ↓ 否
保持3分
```

### 快速打分清单

- [ ] 是否触发0分红线？
- [ ] 是否触发1分条款？
- [ ] 是否触发2分条款？
- [ ] 3分基础要求是否达标？
- [ ] 如果想给4分：能举证"极其突出"吗？
- [ ] 如果想给4分：自检清单全部通过吗？
- [ ] **疑似4分但犹豫 → 一律给3分**
