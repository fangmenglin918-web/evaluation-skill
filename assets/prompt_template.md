### 🌟 4分：惊艳卓越（极度稀缺）

**定义**：千里挑一的卓越表现，提供了**独特的增量价值**。

**⚠️ 4分给分原则（必读）**：
- 4分是"惊艳"，不是"还不错""做得挺好"
- 4分是"超出预期"，不是"符合预期"
- **1000个样本里最多5-10个能拿4分**
- **疑似4分但犹豫 → 一律给3分**

#### "极其突出"的4重测试

1. **替换测试**：删掉这个元素，质量明显下降吗？
2. **惊喜测试**：用户看到会说"哇"还是"嗯"？
3. **举证测试**：能逐字引用哪句话突出吗？
4. **独特性测试**：通用AI能给出类似表达吗？

#### 强制自检（4分必答清单）

| 自检项 | 要求 |
|--------|------|
| Q1: 是否触发任何否决条件？ | 必须"否" |

| Q3: 这句话通过4重测试吗？ | 必须"是" |
| Q4: 和3分标准答案相比，明显优势是什么？ | 必须具体说明 |
| Q5: 如果改成3分，你会删掉哪句话？ | 删不掉→本就是3分 |
| Q6: 加分点是否在3分基准线清单里？ | 必须"不在" |
| Q7: 1000个类似样本里，这个能排前10吗？ | 必须"能" |

- {{红线检查项列表}}

**Step 2: 检查1分（严重缺陷排查）**
- {{1分检查项列表}}
- Hit → 直接1分，结束

**Step 3: 检查2分（瑕疵排查）**
- {{2分检查项列表}}
- Hit → 定为2分，结束

**Step 4: 确认3分（基线达标）**
- 通过0-2分排查 → **自动给3分**
- 检查各维度基础要求是否满足
- **默认停在3分**

**Step 5: 判断4分（极其严格）**
- 先检查否决条件 → 触发则保持3分
- 逐条列出加分点 → 排除3分基准线行为
- 评估是否"极其突出" → 通过4重测试
- 完成强制自检 → 全部正向才能给4分
